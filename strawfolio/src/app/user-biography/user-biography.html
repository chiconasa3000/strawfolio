Ingeniero en Aprendizaje Automático con una exitosa transición desde el desarrollo de software. Mi experiencia como desarrollador de software en MicroData también me ha brindado un profundo conocimiento del ciclo de vida del desarrollo de software, donde diseñé microservicios para la gestión de citas técnicas en el sector automotriz, utilizando tecnologías como Spring, Java y Angular y desarrollé la capacidad de colaborar de manera efectiva con equipos multifuncionales y de trabajar dentro de metodologías ágiles como Scrum.
